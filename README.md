# my_portfolio_site

### <주제>
- jsp를 이용한 답변형 게시판 

<br/><br/>

### <사용한 기술>

- 프론트엔드 : html, css, javascript, bootstrap  
- 백엔드 : java, jsp, mysql, tomcat

<br/><br/>

### <주요 기능>

#### 회원가입 & 로그인  
  - mvc 패턴 이용한 분기별 프로세스 구현  
  - java, jstl, el을 활용해 세션 처리
  
#### 게시판
  - 기본적인 CRUD, 페이징, 댓글, 댓글의 댓글 기능 구현  
  - 로그인 유무에 따라 기능 제한 처리
